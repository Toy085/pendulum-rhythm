# Pendulum Rhythm

A rhythm game where you play as a pendulum! This game uses Love2d & Lua.

## How to Play
1. Download the game from [GitHub Releases](https://github.com/Toy085/pendulum-rhythm/releases) or [Itch.io](https://toy085.itch.io/pendulum-rhythm).
2. Extract (unzip) the `game.zip` file to a folder on your computer.
3. Open the folder and run `game.exe`.

### Controls
Use arrow keys to navigate menus and press space or enter for conformation.

Press space when the pendulum overlaps the notes for a prefect score.
